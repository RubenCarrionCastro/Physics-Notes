# Exercise 1 – Two-Electron Atomic Systems

This exercise analyzes helium-like ions using several approximation methods:
independent particle model, screened potential, perturbation theory and a
simple variational approach.

## Contents
- `Autoevaluacion1Atomica.nb` – Mathematica notebook with all computations.
- `results.pdf` – Complete write-up with derivations, plots and discussion. 
- `graphics/` – Plots exported from Mathematica.

## Methods implemented
- Energy estimation with independent particle model.
- Screening using an effective charge \( Z_{\text{eff}} = Z - S \).
- First-order perturbation theory.
- Variational method using exponential trial functions.
- Computation of relative errors vs exact values.
